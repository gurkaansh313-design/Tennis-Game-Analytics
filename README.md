# Tennis-Game-Analytics
Streamlit dashboard for tennis data analysis using SQL and Python

1)  Overview
This project analyzes tennis player data using SQL and Streamlit.

2) Tech Stack
- Python
- MySQL
- Streamlit
- Pandas
- Plotly

3) Features
- KPI Dashboard
- Leaderboard (Top Players)
- Country-wise Analysis
- Rank Filter
- Search Functionality

4) Key Insights
- USA has highest number of competitors
- Top players dominate rankings
- Rankings show stability among top players

5) How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
